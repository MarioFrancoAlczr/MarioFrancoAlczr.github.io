  • ACCESIBILITY STANDARS

  • Web Content Accessibility Guidelines

    In 1999 the Web Accessibility Initiative, a project by the World Wide Web .Consortium (W3C), published the Web Content Accessibility Guidelines WCAG 1.0. On 11 December 2008, the WAI released the WCAG 2.0 as a Recommendation. WCAG 2.0 aims to be up to date and more technology neutral. Though web designers can choose either standard to follow, the WCAG 2.0 have been widely accepted as the definitive guidelines on how to create accessible websites. Governments are steadily adopting the WCAG 2.0 as the accessibility standard for their own websites.

  • W3C

    The Initiative for Web Accessibility, Web Accessibility Initiative (WAI) in English, is an initiative of the World Wide Web Consortium (W3C) that aims to facilitate access for people with disabilities, developing accessibility guidelines, improving tools for evaluation and repair of Web accessibility, carrying out educational and awareness-raising work in relation to the importance of accessible design of Web pages, and opening new fields in accessibility through research in this area. The WAI works with organizations around the world to develop strategies, guidelines, and create resources and tools that help make the Web accessible to people with disabilities.

  • WCAG 1.0

    Web Content Accessibility Guidelines 1.0 (WCAG 1.0) is a May 5, 1999 W3C recommendation that explains how to make web content accessible to people with disabilities.
    So far, WCAG 1.0 is recognized as the de facto international standard for web accessibility. It has even become the de jure standard in many countries, as in the case of Spain. Unfortunately, WCAG 1.0 can be difficult for the beginner to understand and may seem overwhelming and out of hand due to the breadth of topics it covers. To avoid this situation, the W3C provides the Curriculum for Web Content Accessibility Guidelines 1.0, a tutorial full of explanations and examples.

  • WAI

    The Web Accessibility Initiative (WAI) or Web Accessibility Initiative is a branch of the World Wide Web Consortium that ensures the accessibility of the Web.
    Publish the Web Content Accessibility Guides. The general idea of the WAI is to create a set of clear rules.

    The degree of accessibility is established in levels called A, AA, and AAA, corresponding respectively to minimum criteria of accessibility, extended, and maximum accessibility. Technically, accessibility is implemented through guidelines of structural logic of documents, self-explanatory content and additional semantics, with the intention of allowing the widest possible audience of users with different levels of technological endowment and sensory capacity, to access the information. that is intended to represent and transmit.
